SIGPI – Integrated Academic Project Management System





📖 Project Overview



SIGPI is a web platform for managing and tracking academic integrative projects.



Many schools use emails, shared folders, and different tools to receive project submissions. This makes the process hard to organize and follow.



SIGPI puts everything in one place. Students, teachers, coordinators, and partner companies can use the platform to submit, review, track, and manage projects in a simple and organized way.



SIGPI centralizes academic project submissions, evaluations, audit records, reports, project showcases, and GitHub integration in a single system.







🚀 Key Features



\* Centralized project submissions

\* Recorded feedback and grades

\* Role-based access for different users

\* GitHub repository integration

\* Progress and status dashboards

\* Academic reports

\* Automatic deadline control

\* Connection between students and partner companies
